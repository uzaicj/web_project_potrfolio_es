# Portafolio

Este proyecto es una landing page de portafolio desarrollada con HTML5 y CSS3. El objetivo fue practicar la construcción de layouts modernos como flexbox y grid, la organización de componentes y el diseño responsivo.

La página incluye una sección principal, sección de habilidades, tarjetas de proyectos y un footer de contacto, utilizando Flexbox y Grid para lograr un diseño limpio y adaptable a diferentes tamaños de pantalla.

Durante el desarrollo se puso especial atención en:

el uso de HTML semántico

diseño responsivo mediante media queries

organización del CSS con una estructura tipo BEM

implementación de efectos visuales como gradientes y overlays

Este proyecto fue realizado como parte de mi proceso de aprendizaje en un bootcamp de desarrollo Full Stack, enfocado en fortalecer las bases del desarrollo front-end.
